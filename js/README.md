# PDF 與多媒體播放器整合網站

這是一個整合 PDF 閱讀器和多媒體播放器的網站，支援 PDF 文件、YouTube 影片、MP4 影片和 MP3 音訊播放。

## 功能特色

1. **PDF 閱讀器**
   - 支援線上 PDF 文件閱讀
   - 上下頁瀏覽
   - 縮放功能（放大、縮小、適合寬度）
   - PDF 清單顯示與瀏覽統計

2. **多媒體播放器**
   - YouTube 影片播放
   - 本地 MP4 影片播放
   - MP3 音訊播放
   - 播放控制（播放、暫停、靜音、全螢幕）

3. **清單管理**
   - PDF 清單（data/pdf-manifest.json）
   - 媒體清單（data/media-manifest.json）
   - 可輕鬆新增或移除項目
